# Scrumdinger

Scrumdinger is an iOS app developed with SwiftUI as part of Apple's tutorial that can be found [here](https://developer.apple.com/tutorials/app-dev-training). 

Through the tutorial a fully functional app was built, leveraging SwiftUI's built-in support for accessibility.
